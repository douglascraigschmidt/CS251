Overview of the Array Assignment

The purpose of this assignment is to give you experience using Eclipse to develop a simple Java Array class. An Array is an abstract data type (ADT) with operations such as get(), get(), and size(). Your task is to implement an array of characters in Java. This array will be different from Java built-in arrays since it can be resized.

Graduate students (or students who are taking this course for graduate credit) must do the following additional work:

  • Implement the requirements of the Iterable interface.

As much as possible, you should avoid using loops in this assignment. In our solution, we only needed to use an explicit looping construct once.

The following resources may be helpful in completing this assignment:
